# poet
我是“小诗姬”，全唐诗作为训练数据。可以写押韵自由诗、藏头诗、给定若干字作为主题的诗。
<br /> <br /> 
环境要求：
---
python:3.x <br /> 
tensorflow  1.x

<br /> <br /> 
运行<br /> 
---
运行训练：python train.py <br /> 
---
![image](https://github.com/norybaby/poet/blob/master/doc/train.png)

<br /> <br /> 
 
 
运行写诗服务: python poem_server<br /> 
---
此步需要先运行训练产生好模型后。

客户端用浏览器访问，效果参照：<br />
